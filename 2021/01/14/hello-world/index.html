记录在blog创建过程中出现的问题

首先，当然是blog的创立，我参考了很多，最后决定使用github的github pages服务，具体则是依靠Hexo建立的站点。

github pages可以使用Git来控制，不得不说这是相当方便了。

使用Hexo的过程在这：

npm install -g hexo-cli 在这里可以选择是否局部安装hexo包

hexo init blog

cd blog

npm install

hexo s

就几句简单的命令而已，随后再安装hexo的git安装包部分就完全OK了。
